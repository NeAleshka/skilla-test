# Тестовое задание для Skilla Работа

Разработан раздел звонки CRM системы. 
 
**Функции/features**
* листинг звонков с выборкой по датам
* выбор входящих, исходящих или всех звонков
* проигрывание записи (если есть) 

**Используемые технологии**
* HTML
* CSS
* Grid
* Flex
* Javascript
* React
* Redux
* Redux-saga
